# Sentiment Analysis

A simple NLP project that classifies tweet text into **positive, negative, or neutral** sentiment.

## Steps
- Clean and preprocess text  
- Remove noise (URLs, mentions, punctuation)  
- Apply TF-IDF vectorization  
- Train Logistic Regression model  
- Evaluate accuracy and classification report

## Tech Used
Python · Pandas · NLTK · Scikit-learn

## Result
The model achieved **high accuracy** and performs well in identifying tweet sentiment.
